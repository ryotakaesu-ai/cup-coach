対戦相手

マスター3 白帯フェザー / 同階級1名のみ・決勝戦
→ 公式戦記録なし＝競技経験ほぼ無し。TF系所属＝ガード主体の可能性高

勝負は最初の10秒のグリップファイト 両者引き込み合戦になる確率が高い → 先にグリップ取った方が圧倒的有利。開始合図から能動的に動いて試合を支配する。
タイムライン（5分1本・先生指示）
0:00-0:15 軽くグリップファイト → 即引き込み 0:15-0:30 クローズドへ、襟・袖グリップ確保 0:30-2:00 三角絞めセットアップ ├ 決まる → 一本勝ち └ 防がれる → 腕十字へ移行 2:00-3:30 腕十字セットアップ ├ 決まる → 一本勝ち └ 防がれる → オモプラータ or スイープ 3:30-5:00 スイープで2点取り → トップキープ 理想: 3分以内にサブで決める 最低: スイープ2点取って勝つ
三角→腕十字→オモプラ ABC連携
  • クロス襟＋袖の両グリップ。腰を密着
  • 相手の片腕を脚の間に入れさせる → クロス襟引き＋袖引きで前崩し
  • 三角防がれた → 抱えてる側と反対の腕を抜いて腕十字
  • 腕十字の腕が戻ってきた → そのまま肩に足かけてオモプラ
  • 「三角防御の腕 = 腕十字の的」を脳に刻む
相手に先に引き込まれた時
0:00-0:30 スタンド・グリップファイト 0:30-1:30 トレアナ or ニースライス試行 ├ 通る → パス3点 └ 1分通らなければ → グラビティへ 1:30-3:30 グラビティ位置で削る └ 途中でパス成立 → +3点 3:30-5:00 キープ or ニーオン(+2) / マウント(+4)
パス3択（脳内から消す3択以外）
  • クローズド → 立ち上がってStandup Pass
  • オープン（DLR/ラッソー/蜘蛛） → トレアナ
  • 既に膝が内側 → ニースライス（アンダーフック先取り）
グラビティの落とし穴
  • 圧が抜ける（胸骨を相手に刺す）
  • 手で制圧しようとする（手は補助、体重と位置取りが本体）
  • 足を無理に突破（足は無視して上半身制圧）
  • 相手が苦しがった瞬間に緩める（むしろ強める）
JBJJF 白帯マスター3
  • 5分1本・引き込みポイントなし
  • 足関節は原則禁止（ストレートアンクルロックのみ可）
  • パス3点 / マウント4点 / バック4点 / ニーオン2点 / 横四方0点
  • 横四方で止まると追加点ゼロ → 必ず次の位置へ
  • ストーリング（攻撃意思なき抑え込み）にペナルティ。攻撃意思を見せ続ける
ラスト2分の鉄則
  • 新しい動きをしない（取った位置で時計を見る）
  • 相手のリカバリーだけ防ぐ
  • 残り30秒で相手が無理に動いたら、ニーオンで追加点判断
  • レフェリーの声かけで時間把握（時計見すぎると集中切れる）
減量・計量
  • JBJJFは試合直前計量・Gi込み70.0kg（フェザー）
  • 計量日前夜から水抜き開始
  • 糖質を夕食だけカット、タンパク質1.6g/kg維持
