# 初陣 — 5/17 Cup Coach

白帯マスター3 試合準備のためのパーソナル司令塔（単一HTML、LocalStorage、PWA対応）。

## 機能

- **カウントダウン & ピーキング・フェーズ表示**: 4週プランの進捗バー、残り日数で警告アラート切替
- **ドリルカウンター**: 6つの勝ちパターンドリル（三角ABC / グラビティ / ニースライス / トレアナ / 4大スイープ防御 / ブリッジ＆ロール）の本数管理
- **減量トラッカー**: 体重折れ線グラフ、目標・理想ペースライン、1日あたり必要減量の自動計算
- **練習ログ**: 道場・クラス・疲労度・技・スパー結果を記録
- **ケガ予防ガード**: 直近練習の疲労度から動的に警告生成、5/15以降は新技ゼロ強制リマインド
- **試合プラン**: プランA（引き込み→クローズド→三角→腕十字）／プランB（パス）／JBJJFルールのタブ切替

## 使い方

ブラウザで `index.html` を開くだけ。データはブラウザの LocalStorage に保存されるため、端末をまたいだ同期はなし（JSON書き出し/読み込みでバックアップ可）。
